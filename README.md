# nico-panel（MT4版）

MT4用ワンクリック注文パネル。
BUY/SELL注文・TP/SL自動設定・ドラッグによる一括変更・累計Pips集計などをチャート上のパネルで操作できます。

## 主な機能

- ワンクリックBUY/SELL成行注文
- 注文と同時にTP/SLを自動設定
- TP/SLラインをドラッグするだけで全ポジションに一括反映
- スプレッド・日本時間・次足タイマーをリアルタイム表示
- 保有中＋決済済みの累計Pipsを集計
- パネルをドラッグで自由に移動（位置は自動保存）

## ファイル

- `nico-panel.mq4` — インジケーター本体

## インストール方法

1. `nico-panel.mq4` をダウンロード
2. MT4メニュー →「ファイル」→「データフォルダを開く」
3. MQL4 → Indicators フォルダに貼り付け
4. MT4を再起動（またはF7でコンパイル）
5. ナビゲーター →「nico-panel」をチャートにドラッグ
6. 設定ウィンドウで「自動売買を許可する」にチェック

> ⚠️ MT4ツールバーの「AutoTrading」ボタンが緑色（オン）になっていることを確認してください。
